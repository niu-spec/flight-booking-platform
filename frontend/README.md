# Vue前端项目说明

## 技术栈
- Vue 3 (Composition API)
- Vue Router 4
- Pinia (状态管理)
- Element Plus (UI组件库)
- Axios (HTTP客户端)
- Vite (构建工具)

## 安装步骤

1. 安装依赖
```bash
npm install
```

2. 运行开发服务器
```bash
npm run dev
```

3. 构建生产版本
```bash
npm run build
```

## 项目结构
- src/components/ - 可复用组件
- src/views/ - 页面组件
- src/router/ - 路由配置
- src/store/ - Pinia状态管理
- src/api/ - API接口封装
- src/utils/ - 工具函数

## 主要功能
- 用户注册/登录
- 航班查询
- 订单创建
- 在线支付
- 行程管理
